# Orulak - 자산 가격 비교 시각화 플랫폼

## Context
다양한 자산(아파트, 주식, 금/은/원유, 환율)의 가격 등락을 **하나의 차트에서 동시에 비교**할 수 있는 웹 서비스.
사용자가 비교 항목과 기간을 자유롭게 선택하면, 해당 기간의 변동률(%)을 하나의 그래프로 시각화한다.

## 기술 스택
- **Frontend/Backend**: Next.js (App Router)
- **배포**: Vercel
- **DB**: Supabase (API 캐싱, 자산 메타데이터 저장용)
- **차트**: Recharts 또는 Chart.js
- **인증**: 없음 (비로그인 서비스)

## 상세 기능 정리

### 1. 비교 항목 선택
사용자가 비교하고 싶은 자산을 검색/선택하는 기능

| 카테고리 | 항목 예시 | 데이터 소스 (무료 API) |
|---------|---------|---------------------|
| 아파트 | 잠실엘스, 반포래미안 등 단지별 시세 | 공공데이터포털 (국토교통부 실거래가) |
| 국내 주식 | 삼성전자, SK하이닉스 등 | 한국투자증권 OpenAPI |
| 해외 주식 | AAPL, TSLA 등 | 한국투자증권 OpenAPI |
| 원자재 | 금, 은, 원유 (WTI) | 한국투자증권 OpenAPI (해외선물) |
| 환율 | USD/KRW, EUR/KRW 등 | 한국투자증권 OpenAPI |

- 카테고리별 탭 또는 통합 검색으로 항목 선택
- 선택한 항목은 태그/칩 형태로 표시
- 최대 비교 항목 수 제한 (예: 5~10개)

### 2. 기간 선택
- 프리셋 버튼: 1개월, 3개월, 6개월, 1년, 3년, 5년, 10년
- 커스텀 기간: 시작일 ~ 종료일 직접 선택 (Date Picker)

### 3. 차트 시각화 (핵심 기능)
- **변동률(%) 기준 비교**: 절대 가격이 아닌 시작 시점 대비 변동률로 정규화하여 비교
  - 예: 시작일 기준 0%에서 출발, 이후 등락을 % 표시
- **라인 차트**: 각 항목별 다른 색상의 라인
- **인터랙티브 기능**:
  - 마우스 호버 시 특정 시점의 각 항목 수치 툴팁
  - 줌 인/아웃 (드래그로 구간 확대)
  - 특정 항목 라인 클릭으로 하이라이트/숨기기
- **범례**: 항목명, 색상, 현재 변동률 표시

### 4. 데이터 테이블 (보조)
- 차트 아래에 선택한 항목들의 요약 정보 테이블
  - 항목명 | 시작가 | 현재가 | 변동률 | 최고점 | 최저점

### 5. 공유 기능
- 현재 비교 설정(항목 + 기간)을 URL 쿼리 파라미터로 인코딩
- URL 복사 버튼으로 다른 사람에게 동일한 비교 화면 공유 가능

### 6. 백엔드 / 데이터 관리
- **API Route (Next.js)**: 각 데이터 소스별 API 호출을 통합하는 서버 사이드 엔드포인트
- **Supabase 활용**:
  - API 응답 캐싱 (동일 요청 반복 방지, API 호출 횟수 절감)
  - 자산 메타데이터 저장 (아파트 단지 목록, 종목 코드 매핑 등)
- **데이터 정규화**: 서로 다른 API의 응답 형식을 통일된 형태로 변환

## 구현 순서 (제안)

### Phase 1: 기반 구축
- Next.js 프로젝트 초기화 (App Router, TypeScript, Tailwind CSS)
- Supabase 연동 설정
- 기본 레이아웃 및 UI 컴포넌트

### Phase 2: 데이터 레이어
- 각 데이터 소스 API 연동 (국내 주식 우선)
- API Route 구현
- 데이터 정규화 로직
- Supabase 캐싱 레이어

### Phase 3: 핵심 UI
- 항목 검색/선택 컴포넌트
- 기간 선택 컴포넌트
- 차트 컴포넌트 (변동률 비교)

### Phase 4: 추가 데이터 소스
- 아파트 실거래가 API 연동
- 해외 주식, 원자재, 환율 API 연동

### Phase 5: 마무리
- 공유 기능 (URL 인코딩)
- 데이터 테이블
- 반응형 디자인
- Vercel 배포
